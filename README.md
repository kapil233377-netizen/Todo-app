# Todo App 🚀

A Full Stack Todo Application built with Flask.

## Live Demo

https://todo-app-ai1f.onrender.com

## Features

- User Signup/Login
- Password Hashing
- User-wise Todos
- Add, Update, Delete Todos
- Search Todo
- Complete/Incomplete Toggle
- Dashboard with Progress Tracking
- Responsive UI

## Technologies Used

- Python
- Flask
- SQLite
- SQLAlchemy
- Bootstrap
- HTML/CSS
